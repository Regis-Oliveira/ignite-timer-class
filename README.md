## :key: Getting Started

```bash
# clone the repo:
git clone https://github.com/Regis-Oliveira/ignite-timer-class.git

# open the project:
cd ignite-timer-class

# install the dependencies:
npm install
# or
yarn install

#execute
npm run dev
# or
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
